# PERMISSIONS AND GROUPS

## PERMISSIONS

### can_create
- This means the user can create an article

### can_view
- This means the user can view articles

### can_edit
- This means the user can edit articles

### can_delete
- This means the user can delete articles

## GROUPS

### Admins
- They have access to
1. can_create
2. can_view
3. can_edit
4. can_delete

### Editors
- They have access to
1. can_create
2. can_view
3. can_edit
4. can_delete

### Viewers
- They have access to
1. can_view
